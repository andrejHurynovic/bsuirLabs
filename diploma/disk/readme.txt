1. При присутствии лицензии разработчика Apple.
Данный метод доступен на операционных системах macOS версии 13,0 и выше и Windows версии 10 и выше.

Шаг 1. Войти в личный кабинет разработчика Apple по адресу: https://developer.apple.com.
Шаг 2. Перейти во вкладку Devices.
Шаг 3. Нажать на кнопку с изображением плюса.
Шаг 4. Ввести UDID устройства, на которое планируется установка.
Шаг 5. На операционной система Windows установить iTunes по адресу: https://support.apple.com/ru-ru/HT210384.
Шаг 6. Подключить устройство к компьютеру. 
Шаг 7. На операционной системе Windows зайти в iTunes и выбрать устройство; на операционной системе macOS выбрать устройство в Finder.
Шаг 8. Перетащить файл bsuirSchedule.ipa во вкладку приложения.
Шаг 9. Выполнить синхронизацию устройств.

2. При отсутствии лицензии разработчика.
Данный метод доступен на операционной системе macOS версии 13,0 и выше.

Шаг 1. Установить Xcode по ссылке: https://developer.apple.com/download/all/?q=Xcode.
Шаг 2. Распаковать архивный файл bsuirSchedule.zip системным приложением «Распаковка».
Шаг 3. Открыть файл bsuirSchedule.xcodeproj.
Шаг 4. Подключить устройство к компьютеру.
Шаг 5. Выбрать подключённое устройство в меню «Destination»
Шаг 6. Выбрать меню Product, в нём нажать на Run.
